# Cognisearch
A hackathon project for the Cope Steamworks Coding Club. Users store their notes in plain text (.txt) files, then enter a search query. The system uses a key phrase extraction algorithm to identify important terms from the query and performs a ranked search across all notes to return the most relevant information quickly and efficiently.
